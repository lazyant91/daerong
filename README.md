## 📖 프로젝트 설명
'데롱(Daerong)'은 클립보드에 복사된 이미지(스크린샷 등)를 화면 상단에 즉시 플로팅하고 고정하여, 개발 및 디자인 작업 시 여러 참고 자료를 간편하게 대조하며 작업할 수 있도록 돕는 프리미엄 데스크톱 유틸리티입니다.

### 주요 기능
- **클립보드 감지 플로팅**: 이미지 복사 시 화면에 오버레이 윈도우 생성 및 고정
- **실시간 화면 확대 돋보기 (Live Magnifier)**: 드래그 영역 지정을 통한 실시간 캡처 화면 줌인, 마우스 Panning 이동 및 휠을 이용한 배율 조작 지원
- **통합형 커스텀 타이틀바**: 프레임리스 기반의 콤팩트한 윈도우 타이틀바 구성 및 테마 변경 시 창 제어 오버레이 색상 실시간 동적 연동
- **20종 전역 테마 제공**: 설정 창을 통해 전역 테마 톤앤매너 실시간 적용 및 기억
- **콤팩트 UI 레이아웃**: 자동 크기 조절(Auto-Resizing) 및 스크롤바 없는 콤팩트한 설정 화면 제공
- **트레이 최소화 및 단축키**: 부팅 시 자동 시작 및 트레이 최소화, 사용자 지정 단축키 설정
